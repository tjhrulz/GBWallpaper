GBWallpaper

Gameboy Emulator as windows wallpaper

Download About

Feeling bored with basic picture wallpapers?

Want to play a quick game while all windows are minimzed? Well why not just run a full gameboy emulator as desktop wallpaper!
Don’t want to play? then disable keyboard input & use Macro input, save sequence of moves & automatically play it back.
Don’t worry about performance loss, emulator will pause when running fullscreen application.

Here are some of the feature

Sleeps when Busy

Emulator will pause when running fullscreen application or games - 0% cpu & gpu usage.

Privacy

Keyboard input will get disabled when not on desktop, games will continue to run.

Tons of Feature

Different layouts styles, macro input sequence, audio visualiser.

customizable

Keyboard input & audio can be disabled, Save for supported games.
